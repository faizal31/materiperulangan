while (true) adalah jenis perulangan tak terbatas (infinite loop) yang akan terus berjalan selamanya, kecuali diberi perintah untuk keluar seperti break.

Perulangan ini akan terus mengeksekusi blok kode di dalamnya selamanya karena kondisinya selalu bernilai True. Ini sangat berguna saat Anda ingin kode terus berjalan sampai ada kondisi lain yang memaksanya untuk berhenti, seperti dengan menggunakan break atau return.

âœ… Kapan Digunakan?

  • Menjalankan program terus-menerus sampai ada kondisi tertentu (misalnya input pengguna).
  • Menunggu suatu kondisi eksternal (event).
  • Membuat menu interaktif yang terus muncul hingga user memilih keluar.

ðŸ’¡ Struktur Umum while (true)

while (true)
{
    // Blok kode yang terus diulang
    // Gunakan 'break' untuk keluar dari loop
}

ðŸ’» Contoh Program 1: Perulangan Tak Terbatas dengan break

using System;

class Program {
    static void Main() {
        int i = 0;

        while (true) {
            Console.WriteLine("Perulangan ke-" + i);
            i++;

            if (i >= 5) {
                break;  // Keluar dari loop setelah 5 kali
            }
        }
    }
}

ðŸ’» Contoh Program 2: Menu Interaktif Sederhana

using System;

class Program {
    static void Main() {
        while (true) {
            Console.WriteLine("Menu:");
            Console.WriteLine("1. Cetak Salam");
            Console.WriteLine("2. Keluar");
            Console.Write("Pilih menu (1/2): ");
            string pilihan = Console.ReadLine();

            if (pilihan == "1") {
                Console.WriteLine("Halo, jadi ini adalah contoh menu interaktif sederhana!");
            } else if (pilihan == "2") {
                Console.WriteLine("Program selesai.");
                break;
            } else {
                Console.WriteLine("Pilihan tidak valid. silahkan pilih angka 1 atau 2.");
            }

            Console.WriteLine(); // baris kosong
        }
    }
}
