while (true) adalah jenis perulangan tak terbatas (infinite loop) yang akan terus berjalan selamanya, kecuali diberi perintah untuk keluar seperti break.

Perulangan ini akan terus mengeksekusi blok kode di dalamnya selamanya karena kondisinya selalu bernilai True. Ini sangat berguna saat Anda ingin kode terus berjalan sampai ada kondisi lain yang memaksanya untuk berhenti, seperti dengan menggunakan break atau return.

âœ… Kapan Digunakan?

  • Menjalankan program terus-menerus sampai ada kondisi tertentu (misalnya input pengguna).
  • Menunggu suatu kondisi eksternal (event).
  • Membuat menu interaktif yang terus muncul hingga user memilih keluar.

ðŸ’¡ Sintaks Umum while (true)

while (true)
{
    // Mengeksekusi kode yang ada di dalam loop
}
  • true: Kondisi dalam perulangan yang selalu bernilai benar. Karena nilai true selalu benar, maka perulangan ini akan terus dijalankan tanpa henti.
  • while: Perintah untuk memulai perulangan dengan kondisi tertentu.

ðŸ”„ Cara Kerja:

  • Periksa kondisi: Sebuah while loop akan mengecek kondisi yang diberikan. Dalam hal ini, kondisi true selalu benar, sehingga perulangan akan terus dilakukan
  • Eksekusi kode di dalam loop: Kode dalam blok {} akan dijalankan berulang kali.
  • Iterasi ulang: Setelah selesai menjalankan kode dalam loop, perulangan akan mengecek kondisi lagi. Karena kondisi true, perulangan berlanjut.

ðŸ’¡ Contoh Penggunaan

1. ðŸ’»Contoh Infinite Loop:


Berikut adalah contoh sederhana dari perulangan while (true) yang menunggu input pengguna dan keluar saat mengetik "exit":

using System;

  class Program
  {
      static void Main()
      {
          while (true)
          {
              Console.Write("Ketik 'exit' untuk keluar: ");
              string userInput = Console.ReadLine();
              
              if (userInput.ToLower() == "exit")
              {
                  Console.WriteLine("Keluar dari program.");
                  break;  // Menghentikan loop
              }
              else
              {
                  Console.WriteLine($"Kamu mengetik: {userInput}");
              }
          }
      }
  }

Penjelasan:

  • Console.ReadLine() digunakan untuk menerima input pengguna.
  • Program akan terus meminta input hingga pengguna mengetikkan "exit", yang akan menghentikan perulangan dengan perintah break.
  • break menghentikan loop saat kondisi tertentu terpenuhi.

2. ðŸ’»Contoh Infinite Loop Tanpa Kondisi:


Berikut adalah contoh while (true) tanpa kondisi penghentian yang eksplisit, hanya berhenti ketika program dibunuh atau dihentikan secara eksternal:

using System;

    class Program
    {
        static void Main()
        {
            while (true)
            {
                Console.WriteLine("Program sedang berjalan... tekan Ctrl+C untuk menghentikan.");
                System.Threading.Thread.Sleep(1000);  // Menghentikan eksekusi sementara selama 1 detik
            }
        }
    }

Penjelasan:

  • Program akan mencetak pesan terus menerus setiap detik tanpa henti.
  • Thread.Sleep(1000) menghentikan sementara eksekusi program selama 1000 milidetik (1 detik) di setiap iterasi.

3. ðŸ’» Contoh Program : Perulangan Tak Terbatas dengan break

using System;

class Program {
    static void Main() {
        int i = 0;

        while (true) {
            Console.WriteLine("Perulangan ke-" + i);
            i++;

            if (i >= 5) {
                break;  // Keluar dari loop setelah 5 kali
            }
        }
    }
}

4. ðŸ’» Contoh Program : Menu Interaktif Sederhana

using System;

class Program {
    static void Main() {
        while (true) {
            Console.WriteLine("Menu:");
            Console.WriteLine("1. Cetak Salam");
            Console.WriteLine("2. Keluar");
            Console.Write("Pilih menu (1/2): ");
            string pilihan = Console.ReadLine();

            if (pilihan == "1") {
                Console.WriteLine("Halo, jadi ini adalah contoh menu interaktif sederhana!");
            } else if (pilihan == "2") {
                Console.WriteLine("Program selesai.");
                break;
            } else {
                Console.WriteLine("Pilihan tidak valid. silahkan pilih angka 1 atau 2.");
            }

            Console.WriteLine(); // baris kosong
        }
    }
}

Menggunakan break untuk Menghentikan Infinite Loop

Perulangan while (true) sering digunakan dengan perintah break untuk mengendalikan kapan perulangan dihentikan, seperti pada contoh pertama. Hal ini memberikan fleksibilitas karena kita bisa menentukan kapan program harus berhenti berdasarkan kondisi runtime.

Bahaya Infinite Loop:

Jika tidak digunakan dengan hati-hati, while (true) bisa berbahaya karena perulangan yang tak berujung dapat menyebabkan program menggunakan sumber daya secara berlebihan, menyebabkan aplikasi menjadi tidak responsif, atau bahkan membuat sistem crash jika tidak diberhentikan dengan benar.

Penggunaan dalam program:

While (true) sering digunakan dalam:

  • Server yang terus-menerus menerima permintaan dari klien.
  • Game atau aplikasi real-time yang perlu melakukan pengecekan atau pembaruan secara terus-menerus.
  • Aplikasi berbasis event di mana perulangan berjalan terus hingga program diberhentikan atau diberi perintah untuk berhenti.
